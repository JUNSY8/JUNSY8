## About Me

I build backend systems for business-critical applications — currently focused on **HR/payroll platforms** with complex, regulation-driven logic. My work centers on translating legal/business rules (e.g. Colombian labor law) into clean, maintainable database and application architecture.

**Currently building:**
- 🔧 Overtime/surcharge configuration module (Laravel + Vue) — compliant with **Ley 2466 de 2025**
- 🐍 Python desktop tooling for browser automation and QA (Tkinter + Selenium)
